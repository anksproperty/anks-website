# ANKS Property Management Website

This is a simple static website for ANKS Property Management.

## Files

- `index.html` — homepage
- `contact.html` — contact form page
- `assets/styles.css` — full visual styling
- `assets/script.js` — mobile navigation behavior

## Important before publishing

Replace this placeholder email everywhere:

`hello@ankspropertymanagement.ca`

with the confirmed ANKS email address.

The form currently uses FormSubmit:

`https://formsubmit.co/hello@ankspropertymanagement.ca`

The first form submission usually requires email verification before messages start forwarding.

## GitHub Pages upload

Upload the full folder contents to your GitHub repository root. Your root should contain:

- `index.html`
- `contact.html`
- `assets/`

Then deploy from `main` branch and `/root`.
